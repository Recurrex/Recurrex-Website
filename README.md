 # Next.js Portfolio Starter — คำอธิบายแบบกันเอง (ภาษาไทย)

 นี่คือโค้ดตัวอย่างสำหรับเว็บพอร์ตโฟลิโอที่ทำด้วย Next.js — เหมาะสำหรับเอาไปแก้ไขเป็นเว็บของตัวเองได้เลย

 ถ้าอยากดูวิดีโอสอนหรือโค้ดต้นฉบับ ลิงก์เดิมยังอยู่ใน README ต้นฉบับ แต่ไม่จำเป็นต้องเปิดดูก่อนก็ได้ ถ้าจะลองรันเลยก็ตามคำสั่งด้านล่าง

 ---

 ภาพตัวอย่างจะอยู่ในโฟลเดอร์ `website images/` (มีทั้งภาพโหมดสว่างและมืด)

 ### ไลบรารีที่ใช้ (สั้น ๆ)
 - ฟีเจอร์หลัก: `next`, `react`, `react-dom`
 - ถ้าจะเพิ่มอนิเมชันใช้ `framer-motion`

 ### ทำงานกับโปรเจกต์นี้ (เร็ว ๆ)
 - ติดตั้ง: `npm install`
 - รัน dev server: `npm run dev` แล้วเข้า `http://localhost:3000`
 - สร้าง production: `npm run build` แล้ว `npm run start`

 ถ้าต้องการให้ผมขยาย README เป็นคู่มือการพัฒนาแบบละเอียด (deploy, CI, SEO ฯลฯ) บอกได้เลย ว่าอยากได้ระดับไหน
